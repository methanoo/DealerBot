## 🚀 Cos'è DealerBot

DealerBot è un sistema hardware basato su **ATtiny85** che permette di distribuire carte automaticamente utilizzando telecomandi a infrarossi (protocollo NEC).

Ogni giocatore ha un telecomando → preme il pulsante → il sistema riconosce chi è → assegna la carta.
